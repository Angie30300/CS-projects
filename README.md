# CS-projects
-CS projects I created in highschool AP CSP in Python-

- grade.py: Writes a function that asks the user to input a score as an interger and returns the appropriate letter
- grocery.py: writes functions that checks the prices of multiple items in a grocery store, rounds prices, and provides a list of items within a range.
- hacker.py: Writes a function that finds/searches for a hacker by looking at login history of a website
- peak.py: algorithm is used to identify local maxima within a data series—points that are "higher" or "greater" than the points immediately surrounding them.
- pokemon.py: Simulation of a pokemon evolutuion game
- slot.py: Simulation of a slot machine that can also be altered to give the casino more profit. 
- todo.py: Creates a to do list and allows you to alter it as tasks are done or new ones come up.
- weather.py: Gives recommendations of what to wear according to the weather/tempurature.
